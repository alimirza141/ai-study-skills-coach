Problem:
Students often revise passively and do not know which topics are weak.

Target user:
A GCSE/A-level/university-prep student who wants a simple system to track weak topics and improve over time.

Planned features:

Add study topics
Track confidence score
Track quiz score
Recommend next topic to revise
Generate review questions later using AI
Show progress dashboard later

6-week goal:
Build a working study-tracker app with data storage, dashboard, AI quiz generation, and clear portfolio evidence.
