# Cloud Fundamentals

Cloud computing provides computing resources over the internet.

## IaaS

Infrastructure such as virtual machines, storage and networking.

## PaaS

A managed platform where developers deploy applications without managing most infrastructure.

## SaaS

A complete application delivered through the internet.

## Deployment plan

The AI Study Skills Coach currently runs locally.

The next deployment stages are:

1. Move CSV storage to a database.
2. Create a web interface.
3. Add authentication.
4. Deploy the application to a managed platform.
5. Add monitoring and analytics.
